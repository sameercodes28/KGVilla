## Administrative Costs - Turn-Key Model

| Item | Itemized Price | Turn-Key Price | Notes |
|------|---------------|----------------|-------|
| **Building Permit (Bygglov)** | 40,000 kr | *Bundled* | |
| **KA-kontroll** | 35,000 kr | *Bundled* | |
| **Project Management** | 40,000 kr | *Bundled* | |
| **TOTAL** | 115,000 kr | 0 kr | Absorbed in overhead |

### Explanation
In JB Villan's **totalentreprenad** (turn-key) model, administrative costs are part of the package price, not billed separately. These costs still exist but are absorbed into JB Villan's business overhead.

This is a key difference vs. general contractor quotes which itemize these costs.

## Prefab Construction Efficiency

| Category | GC Price | JB Price | Savings | Type |
|----------|----------|----------|---------|------|
| Exterior walls (per m2) | 3,800 kr | 2,800 kr | 26% | PREFAB |
| Roof (per m2) | 2,200 kr | 1,800 kr | 18% | PREFAB |
| Interior walls (per m2) | 1,450 kr | 1,200 kr | 17% | PREFAB |
| Site overhead | 5% | 3% | 40% | STREAMLINED |
| Contingency | 10% | 6% | 40% | STANDARDIZED |

### Why Prefab is Cheaper
1. **Factory manufacturing** - Controlled environment, no weather delays
2. **Precision cutting** - Less material waste (optimized nesting)
3. **Assembly line efficiency** - Workers specialize, faster production
4. **Reduced site labor** - Crane sets walls in hours vs days of stick-framing
5. **Shorter project duration** - Lower rental costs (scaffolding, containers)
6. **Predictable process** - Fewer change orders, lower contingency needed
